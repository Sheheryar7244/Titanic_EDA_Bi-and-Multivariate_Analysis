🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

📌 Project Overview
This project performs detailed Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns and factors affecting passenger survival.
The analysis includes univariate, bivariate, and multivariate exploration using statistical summaries and visualizations.
---
📊 Key Objectives
- Understand survival distribution
- Analyze impact of gender, class, age, and fare
- Identify relationships between multiple variables
- Detect trends and patterns in passenger data
---
🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
---
📈 Analysis Performed

🔹 Univariate Analysis
- Count plots
- Histograms
- Distribution plots

🔹 Bivariate Analysis
- Survival vs Gender
- Survival vs Passenger Class
- Scatter plots (Age vs Fare)
- Box plots

🔹 Multivariate Analysis
- Pair plots
- Correlation matrix
- Pivot tables
- Heatmaps
---
📊 Visualizations Included
- 📊 Count Plot
- 📉 Histogram
- 📦 Box Plot
- 🔵 Scatter Plot
- 🔗 Pair Plot
- 🔥 Heatmap (Correlation Matrix)
- 📋 Pivot Tables
